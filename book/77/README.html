深入理解 Neutron -- OpenStack 网络实现

Neutron 是 OpenStack 项目中负责提供网络服务的组件，它基于软件定义网络的思想，实现了网络虚拟化下的资源管理。

本书将剖析 Neutron 组件的原理和实现。

在线阅读：GitBook 或 Github。
