scikit-learn 是基于 Python 语言的机器学习工具。

  1. 简单高效的数据挖掘和数据分析工具
  2. 可供大家在各种环境中重复使用
  3. 建立在 NumPy ，SciPy 和 matplotlib 上
  4. 开源，可商业使用 - BSD许可证
